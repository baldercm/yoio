## Basic usage
To download all NPM and Bower dependencies run:

    ./setup.sh

or:

    npm install -g cordova ionic
    cd ~/git/yoio
    npm install
    bower install
    ./node_modules/grunt-protractor-runner/node_modules/protractor/bin/webdriver-manager update
